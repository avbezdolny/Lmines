# MINES (sapper)

Open all the cells on the field except those containing mines. The number in the open cell indicates the number of mines in the adjacent cells. You can mark a cell with a flag if you think it is mined. A cell with a flag is blocked from being opened accidentally. Use tapping on numbers to recursively chord flags and open adjacent cells. Good luck, sapper!

# МИНЫ (сапер)

Откройте все ячейки на поле, кроме содержащих мины. Число в открытой ячейке означает количество мин в соседних ячейках. Можно пометить ячейку флагом, если считаете, что она заминирована. Ячейка с флагом заблокирована от случайного открытия. Используйте нажатия на числа для рекурсивного аккорда флагами и открытия соседних ячеек. Удачи, сапер!
